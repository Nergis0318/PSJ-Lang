# ParkSangjin Lang

`parksangjin-lang`은 `중국인`으로 시작해서 `게이 박상진`으로 끝나는 작은 난해한 프로그래밍 언어 인터프리터입니다.

현재 구현은 Python 패키지로 제공되며, `.psj` 파일을 읽어 한 줄씩 실행합니다.

## 요구 사항

- Python 3.13 이상
- 패키지 실행 도구: `python`, `pip` 또는 `uv`

## 설치

개발 중인 저장소에서 바로 설치하려면 다음 명령을 사용합니다.

```powershell
pip install -e .
```

`uv`를 사용한다면 다음처럼 실행할 수 있습니다.

```powershell
uv sync
uv run parksangjin-lang examples\hello.psj
```

## 실행

설치 후에는 콘솔 명령으로 `.psj` 파일을 실행합니다.

```powershell
parksangjin-lang examples\hello.psj
```

설치하지 않고 저장소에서 바로 실행하려면 `src`를 `PYTHONPATH`에 추가합니다.

```powershell
$env:PYTHONPATH = "src"
python -m parksangjin_lang examples\hello.psj
```

예상 출력:

```txt
HELLO
```

## 프로그램 구조

모든 프로그램은 첫 줄과 마지막 줄이 정해져 있습니다.

```txt
중국인
...
게이 박상진
```

첫 줄이 `중국인`으로 시작하지 않거나 마지막 줄이 `게이 박상진`이 아니면 `SyntaxError`가 발생합니다.

## 값 표현식

표현식은 정수를 만듭니다.

- `.`: `+1`
- `,`: `-1`
- `바`: 변수 참조
- 공백: 곱셈 구분자

예를 들어 `...`는 `3`, `,,`는 `-2`, `... ....`는 `3 * 4`로 계산됩니다.

표현식 안에서 변수는 `바`의 개수로 참조합니다.

- `바`: 0번 변수
- `바바`: 1번 변수
- `바바바`: 2번 변수

대입과 입력에서 대상 변수는 명령 앞쪽의 `바` 개수로 정합니다.

- `박...`: 0번 변수에 `3` 저장
- `바박....`: 1번 변수에 `4` 저장
- `진?`: 입력값을 0번 변수에 저장
- `바진?`: 입력값을 1번 변수에 저장

## 명령 문법

| 기능      | 문법                  | 설명                                                                       |
| --------- | --------------------- | -------------------------------------------------------------------------- |
| 변수 대입 | `{대상}박{표현식}`    | 대상 변수에 표현식 값을 저장합니다.                                        |
| 입력      | `{대상}진?`           | 표준입력에서 정수를 읽어 대상 변수에 저장합니다.                           |
| 정수 출력 | `진{표현식}!`         | 표현식 값을 정수로 출력합니다.                                             |
| 문자 출력 | `진{표현식}ㅋ`        | 표현식 값을 문자 코드로 변환해 출력합니다. 값이 0이면 줄바꿈을 출력합니다. |
| 조건 실행 | `게이{표현식}?{명령}` | 표현식이 0이면 뒤의 명령을 실행합니다.                                     |
| 점프      | `상{표현식}`          | 지정한 줄 번호로 이동합니다.                                               |
| 종료      | `화이팅!{표현식}`     | 표현식 값을 출력하고 프로그램을 종료합니다.                                |

## Python API

패키지 내부에서 직접 인터프리터를 사용할 수도 있습니다.

```python
from parksangjin_lang import ParkSangjin

code = """중국인
진........................................................................ㅋ
게이 박상진"""

ParkSangjin().compile(code)
```

파일 경로로 실행하려면 `compilePath()`를 사용할 수 있습니다.

```python
from parksangjin_lang import ParkSangjin

ParkSangjin().compilePath("examples/hello.psj")
```
